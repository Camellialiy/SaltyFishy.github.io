栈与队列

栈

栈（Stack）是一种特殊的线性表，与一般的线性表相比，栈只允许在表尾进行插入与删除操作。

栈顶（Top）指的是栈的头部，也就是最后一个入栈的元素,也意味着当前的进出栈操作所在的区域

栈底（Bottom）指的是栈的尾部，也就是第一个入栈的元素

空栈，什么元素也没有的栈

栈的特点可以概括为先进后出，这也是栈与队列的关键区别

补充

C 库函数realloc 尝试重新调整之前调用 malloc 或 calloc 所分配的 ptr 所指向的内存块的大小。

函数原型

  • ptr — 指针指向一个要重新分配内存的内存块，该内存块之前是通过调用 malloc、calloc 或 realloc 进行分配内存的。如果为空指针，则会分配一个新的内存块，且函数返回一个指向它的指针。

  • size — 内存块的新的大小，以字节为单位。如果大小为 0，且 ptr 指向一个已存在的内存块，则 ptr 所指向的内存块会被释放，并返回一个空指针。

循环队列

对于数组构建的队列，我们需要特殊考虑把一个问题：由于长度是写死的，那么我们在不断插入，不断删除队列元素后(数量足够多)，最终整个元素的主体一定会向tail端移动，最终front=tail并且指向末尾，这时候如果我们不重置队列，那么这个队列就无法再加入新的元素了。由此，引入循环队列：

循环队列：与一般的队列相同，但是对front与tail的操作做了取余数的操作

比如：当队首指针Q.front=MaxSize-1后，再前进一个位置就自动到0 ，这可以利用除法取余运算（%）来实现。

定义、初始化与一般的队列相同

但是这样就有一个新问题，如何判空？由于此时队满与队空都是front= tail，所以我们就需要一些其他的操作来处理判空

方法1：最简单的方法就是直接加入一个长度成员Q.size
