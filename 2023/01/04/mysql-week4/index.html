sql-learning in week4(using MySQL)

DQL语言学习

进阶3 排序查询

引入:


语法:
select 查询列表 from 表名 （可写条件查询条件） order by 查询列表 排序列表(asc低到高/desc高到低);
特点:
  1. 不写默认升序
  2. asc升序，desc降序
  3. 支持单个字段，多个字段，表达式，函数，别名
  4. 一般放在查询语句的最后面

案例:

  1. 按升降序排序
  2. 按表达式排序
  3. 按别名排序
  4. 按函数排序
  5. 按多个字段排序
